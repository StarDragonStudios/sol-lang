# Sol

Sol is a compiled programming language designed for clarity, strong typing, memory safety, and systems programming.

## Current status

Sol is in early design and bootstrap development.

The first milestone, Sol 0.1, focuses on a procedural language with:

- Static typing
- Functions
- Variables and constants
- Conditions and loops
- Console input/output
- Text file input/output
- A Java bootstrap compiler
- A C runtime backend

## Repository layout

- `spec/`: language specifications and design decisions.
- `compiler/`: Java bootstrap compiler.
- `runtime-c/`: initial C runtime.
- `examples/`: Sol example programs.
- `tests/`: compiler and runtime tests.
